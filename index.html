How it Works

Step 1

Bootstrap straight from your web app

Copy and paste your web page url into ToDesktop. Customise your app design, app icon and window frame UI with no code.

  • Multiple Window
  • Menubar/tray menus
  • Offline support
  • Customise menus
  • Launch on startup
  • Tabs (Mac only)
Step 2

Add desktop code to customise your web app

No need to create a new repository, just add code from the desktop APIs to codebase and get access to system level functionality.

  • Native notifications
  • Global keyboard shortcuts
  • Application menu
  • Context menu
  • Dock/Icon
  • Isolated browser views
  • File system access
  • …plus more APIs in our docs
Step 3

Publish

One-click will publish your desktop app to your customers and give you a download link to put on your website.

  • Super-fast global CDN
  • Download links on your domain
  • Magic universal links
  • Download analytics
  • Manage version numbers
  • Native installers for all platforms
